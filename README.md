# xxh-plugin-prerun-dotfiles

[xxh][] entrypoint for my [dotfiles][].

## Install

Install from GitHub:

```
xxh +I xxh-plugin-prerun-dotfiles+git+https://github.com/chitoku-k/xxh-plugin-prerun-dotfiles
```

Connect:

```
xxh myhost +if
```
